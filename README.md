# StaticTest

Application Java testant les champs static et non-static.

## Description
Ce projet illustre la différence entre un compteur static
et un compteur non-static en Java.

## Résultat attendu
(1,4)
- nbLocal = 1 (propre à chaque instance, repart à 0)
- nbInstances = 4 (partagé entre toutes les instances)

## Concepts utilisés
- Champs static vs non-static
- Constructeurs
- Instanciation d'objets

## Comment exécuter
1. Cloner le projet
2. Ouvrir dans IntelliJ IDEA
3. Exécuter Main.java
